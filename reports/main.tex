% Introduction
\section{Introduction} \label{sec:introduction}

This project falls under option 1: practical implementation of pre-existing algorithm(s). The goal is to implement and evaluate algorithms from two complementary sub-fields of adversarial motion planning. The first is \emph{deceptive motion planning}, which is concerned with generating trajectories that conceal the agent's true objective from an observer \cite{nichols2022adversarial, xu2020single}. The second is \emph{goal identification and interception}, which seeks to infer a observed agent's hidden objective and plan a trajectory to intercept it \cite{zeng2023recognition, tastan2012learning}. Rather than studying these in isolation, the project will pit a deceptive planner against an interception-focused planner in a shared environment, providing a direct adversarial evaluation of each algorithm's strengths and limitations.

% High Level
\subsection{High-Level Motivation} \label{sec:motivation/high-level}

Deceptive motion planning has direct applications in video game AI, where non-player characters must exhibit unpredictable yet purposeful behavior \cite{tastan2012learning, zeng2023recognition}. However, the relevance of this problem extends well beyond entertainment into real-world domains such as strategic logistics, privacy-preserving navigation, and military operations. The ability to move deceptively can be the difference between mission success and failure \cite{xu2020single, nichols2022adversarial}, while the complementary problem of identifying and intercepting deceptive agents is equally critical: security and defense systems must be capable of recognizing when an observed agent is behaving deceptively and responding accordingly \cite{zeng2023recognition, xu2019goal}. Studying the interplay between deception and counter-deception will provide insight into the robustness of each approach and reveals failure modes that would not surface when evaluating either in isolation.

It is important to note that prior work has explored this coupling in the context of autonomous vehicles \cite{espinoza2022deep} and multi-agent navigation \cite{netter2024motion, lecleach2021lucidgames}, but an adversarial pitting in a strategy-game context remains underexplored.

% Methodology
\section{Methodology} \label{sec:methodology}

The project will be implemented in Python, chosen for its extensive ecosystem of machine learning and scientific computing libraries. In particular, I plan to use JAX (via the Equinox library) as the primary framework for differentiable programming, automatic differentiation, and GPU-accelerated computation.

The core of the project involves implementing two competing agents. The first agent will employ a deceptive motion planning algorithm inspired by approaches such as Adversarial RRT* \cite{nichols2022adversarial} and entropy-based deceptive path planning \cite{xu2020single, xu2019goal}, which generate trajectories that minimize an observer's ability to infer the agent's true goal. The second agent will implement an identification and interception strategy, drawing on techniques from inverse reinforcement learning and game-theoretic prediction \cite{zeng2023recognition, tastan2012learning, lecleach2021lucidgames}, with the objective of recognizing the deceptive agent's true goal and planning an intercept trajectory in real time. The approach of coupling adversary identification with reactive planning follows the framework proposed by Netter and Vamvoudakis \cite{netter2024motion}, adapted to the adversarial evaluation setting.

Of note, despite several of the referenced algorithms being originally developed for three-dimensional environments (e.g., first-person shooter game maps \cite{tastan2012learning}), I will be generalizing down to a simplified two-dimensional grid for ease of development and in the interest of having a finished product before the end of the semester. The simulation environment will be custom-built in Python, providing full control over the experimental setup and enabling systematic evaluation of each algorithm's performance under varying conditions.
